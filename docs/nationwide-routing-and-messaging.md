# Маршруты России и сервер сообщений

## Почему нельзя зашивать всю Россию в приложение

Полный адресный граф России: регионы, города, села, улицы, дома, корпуса, дробные номера, подъезды, точки интереса и дорожные маршруты. Это должно жить на сервере и регулярно обновляться из геокодера/картографического провайдера. Мобильное приложение должно хранить только легкий локальный пакет для MVP-зоны и обращаться к серверу по геолокации пользователя.

## Как устроено сейчас

- `src/data/salavatDistrict.ts` содержит детальный локальный пакет для Салаватского района: населенные пункты, улицы, дома-примеры и важные точки.
- В полях "Откуда" и "Куда" поиск умеет подсказывать улицы и дома вроде `Коммунистическая улица, дом 65/1`.
- В заказе есть кнопка "Определить мое место": приложение запрашивает геолокацию и выбирает ближайший адресный контекст.
- `.env.example` содержит будущие адреса серверов:
  - `EXPO_PUBLIC_ADDRESS_SEARCH_URL`;
  - `EXPO_PUBLIC_ROUTE_SEARCH_URL`;
  - `EXPO_PUBLIC_MESSAGE_SERVER_URL`.

## Целевая серверная схема

1. Клиент открывает заказ.
2. Приложение запрашивает геолокацию.
3. Сервер получает координаты и строку поиска.
4. Сервер возвращает ближайшие адреса, дома, подъезды, организации и популярные маршруты.
5. Приложение показывает подсказки, а после выбора маршрута сервер считает цену, расстояние и время.

## Сервер сообщений

Чат поддержки уже выделен в отдельный экран. Пока сообщения сохраняются локально в состоянии приложения, но UI готов к серверу сообщений:

- категории обращений;
- быстрые сообщения;
- история диалога;
- автоответ поддержки;
- отображение режима "локальный канал" или "сервер сообщений".

Продакшен-следующий шаг: подключить WebSocket или SSE для новых сообщений, REST для истории обращений и статусы доставки/прочтения.

## Что важно для бизнес-модели подписки

Если водитель платит за месяц доступа, сервис обязан давать поток заказов. Поэтому маршруты и геолокация должны работать не как статичный справочник, а как трафиковая система: где пользователь находится, какие поездки рядом, сколько водителей доступно и как быстро заказ будет принят.
