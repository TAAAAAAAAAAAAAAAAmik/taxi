# Салаватский район: адресный MVP

Текущий фокус приложения - Салаватский район Республики Башкортостан. Районный центр - село Малояз. Для первой зоны спроса важны Малояз, Янгантау, Кургазак, Мурсалимкино, Аркаулово, Лаклы и остальные населенные пункты района.

## Что загружено

- `src/data/salavatDistrictStreets.ts` - generated-справочник улиц/дорог.
- `scripts/import-salavat-streets.mjs` - read-only импорт из OpenStreetMap Overpass по relation `398510`.
- `src/data/salavatDistrict.ts` - бизнес-логика поиска, ручные POI и популярные маршруты.
- В generated-файле сейчас 493 записи улиц/дорог по 64 населенным пунктам.
- Каждая запись хранит населенный пункт, улицу, источник, алиасы и, если есть, центроид из OSM.

## Источники

Основной импорт MVP взят из OpenStreetMap named highways внутри relation `398510`. Структура данных уже поддерживает `gar`, `osm` и `merged`, поэтому следующий шаг - серверный импорт официальной выгрузки ГАР/ФИАС ФНС и слияние с OSM.

Мы не скрейпим Яндекс, 2ГИС и другие навигаторы без API и лицензии. Для открытого слоя используем OSM/Overpass, для официального адресного реестра - ГАР/ФИАС.

## Как работает поиск

- Сначала ищутся точные совпадения по населенному пункту и улице.
- Затем совпадения по алиасам и частям запроса.
- Для запроса с домом создается точная подсказка из найденной улицы, например `Малояз Коммунистическая 65/1`.
- Ручные POI остаются отдельным слоем: администрация, центр Малояза, автостанция, ЦРБ, музей Салавата Юлаева, Янгантау, Кургазак.

## GPS

Координаты не показываются пользователю и не подставляются в поле адреса. Клиент получает точку только внутренне, после чего вызывает серверный reverse geocoder из `EXPO_PUBLIC_REVERSE_GEOCODE_URL`.

Ожидаемый ответ сервера:

```json
{
  "displayAddress": "Республика Башкортостан, Салаватский район, с. Малояз, Коммунистическая улица, 65/1",
  "region": "Республика Башкортостан",
  "district": "Салаватский район",
  "settlement": "Малояз",
  "street": "Коммунистическая улица",
  "house": "65/1"
}
```

Если reverse geocoder недоступен, приложение показывает: `Не удалось определить адрес, введите вручную.`

## Что должно жить на сервере

- Импорт ГАР/ФИАС и слияние с OSM.
- Reverse geocode по координатам.
- Адресный поиск по всей России.
- Построение маршрутов и ETA.
- Проверка активных водителей, подписок и юридического допуска.
- Сервер сообщений для поддержки и статусов поездок.
